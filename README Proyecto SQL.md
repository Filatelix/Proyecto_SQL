## PROYECTO SQL



#### Crear el esquema de la Base de Datos



Hacer click en el icono de Diagrams que aparece en el navegador de DBeaver abajo a la derecha. Seleccionar la opción de Crear Nuevo Diagrama. Seleccionar la Base de Datos del proyecto, y todas las tablas que están incluidas. Asignar un nombre al esquema, en este caso "Esquema\_Proyecto\_SQL" y finalizar. El fichero erd se ha añadido al repositorio con la resolución del proyecto.



#### Resolución de las consultas propuestas



Abrir un nuevo script de SQL. En el menú Archivo, seleccionar Guardar Como. Grabarlo como fichero sql con el nombre "Script\_Proyecto\_SQL". En este script está el código de cada consulta propuesta de forma consecutiva. Antes de cada código se ha puesto como comentario en número de la consulta y el enunciado. El fichero sql se ha añadido al repositorio de la resolución del proyecto.
